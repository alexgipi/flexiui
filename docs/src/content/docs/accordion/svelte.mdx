## Introduction

The Flexi UI Accordion component enables users to toggle content visibility, offering an organized and customizable solution for collapsible sections in Svelte projects.

## Installation

To install the `Accordion` component in your Svelte project, you can use npm or yarn.

```bash
npm install @flexiui/svelte-accordion
```
