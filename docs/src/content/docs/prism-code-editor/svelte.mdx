## Introduction
The Prism Code Editor component for Svelte provides a syntax-highlighted, editable code editor using Prism.js.

It supports multiple languages, customizable themes, and is ideal for embedding live code examples in your Svelte apps.

## Installation

To install the `PrismCodeEditor` component in your Svelte project, use npm or yarn:

```bash
npm install @flexiui/svelte-prism-code-editor
```

## Languages

The `language` prop at the moment accepts the following five values representing different languages:

- `html`
- `css`
- `javascript`
- `typescript`
- `json`
