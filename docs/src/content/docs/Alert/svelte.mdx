## Introduction
Alerts give users brief and potentially time-sensitive information in an unobtrusive manner.

The Flexi UI Alert component includes several props for quickly customizing its styles to provide immediate visual cues about its contents.

## Installation

To install the `Alert` component in your Svelte project, you can use npm or yarn.

```bash
npm install @flexiui/svelte-alert
```
